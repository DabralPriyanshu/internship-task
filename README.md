# 1 -> I have used atomic design pattern that helps to organize components we divided components into 3 categories atoms molecules and organisms atoms contains small reuseable components molecules are made by using atoms and organism are made by using molecules

# 2 ->In this project i have used atoms and molecules because this is small project and there is no component like a organism

# 3 -> The App components is container which contain all the other components

# 4 -> Card ImageCard InfoCard,Navbar comes under atoms

# 5 -> Hero ,Features ,ImageContainer and InfoSection comes under molecules

# 6 ->All the images are in asset folder can downloaded from https://unsplash.com/

# 7 -> I have put significant effort into mimicking the original design as closely as possible.I have used clear comments throughout the project to explain the logic behind specific sections and styles.

# 8 -> I have used semantic tags for better SEO
